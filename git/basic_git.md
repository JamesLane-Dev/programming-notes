cd into folder of project.
cd <project-folder>

ls to confirm location and contents of folder.

Initialize Git
git init

Add and commit

git add .
git commit -m "Initial commit"

Create Github repo

goto github 
Click new repository
name it.

return to terminal
Add remote

git remote add origin https://github.com/<username>/<repo>.git

Check it

git remote -v

Push

git push -u origin main
